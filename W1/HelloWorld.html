Binary Search Tree (BST)

What is a Binary Search Tree?

A Binary Search Tree is a data structure that maintains sorted data through a hierarchical tree structure. Each node has at most two children (left and right), and follows the property: all values in the left subtree are less than the node's value, and all values in the right subtree are greater.

Key Properties

  • Balance: A balanced BST provides optimal search performance
  • Search Time: O(log n) for balanced trees, O(n) for unbalanced
  • Insertion & Deletion: Maintain the BST property
  • In-order Traversal: Produces sorted sequence
