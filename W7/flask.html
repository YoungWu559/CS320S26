Flask is a lightweight, open-source web framework for Python. It's designed to make building web applications simple and flexible, allowing developers to create both small and large-scale applications with minimal overhead.

Key Features

  • Easy to learn and use
  • Built-in development server
  • RESTful request dispatching
  • Secure cookies support
  • Extensible with plugins
